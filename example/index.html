项目介绍

DevServer 是一个基于 Node.js 的开发服务器工具，提供智能文件监听和自动刷新功能。它能够显著提升前端开发效率，支持局部刷新和完整页面重载。

热重载

文件修改后自动刷新浏览器，无需手动操作

智能刷新

CSS文件变更时仅刷新样式，JS文件变更时智能重载

可配置

支持端口、CORS、静态目录等多种配置选项

跨平台

支持 Windows、macOS 和 Linux 系统

使用方法

基本命令

node . example/index.html example --log-level=debug

带参数的完整命令

node . example/index.html example --port=8080 --open-browser=true --cors=true --full-reload=false --static-dirs=assets,images --start-port=6251 --log-level=info

参数说明

参数 说明 默认值 示例
<html-file> 目标HTML文件路径 必填 example/index.html
<watch-paths> 监听变化的文件或目录 HTML文件所在目录 example src
--port 服务器端口 6251 --port=8080
--open-browser 自动打开浏览器 true --open-browser=false
--cors 启用CORS跨域支持 true --cors=false
--full-reload 启用完整页面刷新 false --full-reload=true
--static-dirs 额外静态资源目录 空 --static-dirs=assets,images
--start-port 自动分配端口的起始值 6251 --start-port=7000
--log-level 日志输出级别 info --log-level=debug

API 说明

DevServer 类

核心服务器类，提供开发服务器功能

import { DevServer } from './dist/index.js';

const config = {
  htmlPath: 'example/index.html',
  watchPaths: ['example'],
  port: 6251,
  openBrowser: true,
  cors: true,
  fullReload: false,
  staticDirs: ['assets']
};

const server = new DevServer(config);
server.start().then(() => {
  console.log('Server started!');
});

DevServerManager 类

自动管理多个服务器实例，支持端口自动分配

import { DevServerManager } from './dist/index.js';

const manager = new DevServerManager({
  startPort: 6251,
  logLevel: 'info'
});

// 自动分配端口（未指定端口时）
const server1 = await manager.start({
  htmlPath: 'example/index.html',
  watchPaths: ['example']
});

// 指定端口
const server2 = await manager.start({
  htmlPath: 'another/index.html',
  watchPaths: ['another'],
  port: 8080
});

// 停止服务器
await manager.stop(server2.config.port);
await manager.stopAll();
